Getting Started with LlamaVault

LlamaVault is a secure credential management system designed for AI applications. This guide will help you get started with installing and using LlamaVault in your projects.

Prerequisites

Before you begin, make sure you have the following:

  • Python 3.8 or higher
  • pip (Python package installer)
  • Basic knowledge of command-line operations

Installation

Install LlamaVault using pip:

pip install llamavault

To verify the installation, run:

llamavault --version

You should see the current version of LlamaVault displayed.

Creating a Vault

Before you can store and retrieve credentials, you need to create a vault:

llamavault init

You'll be prompted to create a master password. This password is used to encrypt and decrypt your vault, so make sure it's strong and you remember it!

Important: If you lose your master password, you will not be able to recover your credentials.

By default, the vault file is created in ~/.llamavault/vault.json. You can specify a different location using the --vault-path option:

llamavault init --vault-path /path/to/my/vault.json

Storing Credentials

To add a credential to your vault:

llamavault add KEY VALUE

For example:

# Add an API key
llamavault add api_key "my-secret-api-key"

# Add a database password
llamavault add db_password "super-secret-password"

# Add complex credentials as JSON
llamavault add database '{"host": "localhost", "port": 5432, "username": "admin", "password": "secret"}'

Retrieving Credentials

To retrieve a credential from your vault:

llamavault get KEY

For example:

# Get an API key
llamavault get api_key

# Get database credentials
llamavault get database

Listing Credentials

To see all the credentials in your vault (without showing their values):

llamavault list

Using LlamaVault in Scripts

LlamaVault provides a Python API for accessing credentials in your scripts:

import llamavault

# Create a vault instance (will prompt for password if not provided)
vault = llamavault.Vault()

# Get a credential
api_key = vault.get("api_key")

# Use the credential
print(f"Using API key: {api_key}")

# Get complex credentials
db_creds = vault.get("database")
print(f"Connecting to {db_creds['host']}:{db_creds['port']} as {db_creds['username']}")

Using Environment Variables

You can export credentials as environment variables for use with applications:

llamavault export api_key

# Or export with a custom environment variable name
llamavault export database_password --env-name DB_PASSWORD

In a script, you can load credentials as environment variables:

import llamavault
import os

# Load credentials into environment variables
vault = llamavault.Vault()
vault.export_to_env("api_key")
vault.export_to_env("database_password", env_name="DB_PASSWORD")

# Use the environment variables
api_key = os.environ.get("API_KEY")
db_password = os.environ.get("DB_PASSWORD")
