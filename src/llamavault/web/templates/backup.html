Backup Vault

Create a backup of your vault for safekeeping.

Important

Backups are encrypted with your vault password. If you forget your password, you won't be able to restore your credentials from a backup.

The backup will be created in the default backup directory. You can use the command line for more control over backup location.

Back
Command Line Alternative

For more control over your backup, use the command line:

llamavault backup --backup-dir /path/to/backup
Security Note

Backups contain your encrypted credentials. While the sensitive values remain encrypted, it's still recommended to keep backups in a secure location.
